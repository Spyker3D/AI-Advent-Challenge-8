# Чат

Введите текст и отправьте его на экране чата. Во время запроса показывается загрузка. Новый диалог создаётся действием нового чата; существующий можно продолжить из локальной истории. При ошибке повторите запрос один раз.

Ответ может отсутствовать из-за сети, rate limit, timeout, недоступности выбранного AI-провайдера или пустого ответа. Пустой ответ следует повторить один раз либо переформулировать. AI может ошибаться; важные сведения нужно проверять. Support Assistant не нажимает кнопки и не меняет настройки за пользователя.
