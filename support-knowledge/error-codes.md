# Демонстрационные диагностические коды CRM

Эти коды принадлежат демонстрационной CRM и не заявлены как коды production backend.

| Код | Значение | Рекомендация |
|---|---|---|
| NETWORK_UNAVAILABLE | На устройстве отсутствует интернет | Проверить Wi-Fi или мобильную сеть |
| OPENAI_RATE_LIMIT | AI-сервис временно ограничил запросы | Подождать и повторить позже |
| OPENAI_TIMEOUT | Ответ не получен вовремя | Повторить один раз |
| OPENAI_UNAVAILABLE | AI-сервис временно недоступен | Повторить позже |
| EMPTY_AI_RESPONSE | Сервис вернул пустой ответ | Повторить или изменить формулировку |
| MCP_UNAVAILABLE | MCP-сервис недоступен | Использовать общую документацию и предложить оператора |
| RAG_INDEX_MISSING | База знаний не подготовлена | Построить индекс; пользователю предложить оператора |
| LOCAL_HISTORY_UNAVAILABLE | История не загрузилась | Перезапустить приложение и проверить локальное хранилище |
| UNKNOWN_ERROR | Неизвестная ошибка | Зафиксировать детали и передать оператору |
