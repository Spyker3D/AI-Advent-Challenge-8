# 1. О проекте

Android AI Assistant — мобильное приложение, реализующее современного AI-ассистента с использованием больших языковых моделей (LLM). Приложение предоставляет пользователю интерфейс для общения с искусственным интеллектом и поддерживает расширяемую архитектуру, позволяющую подключать различные модели, инструменты и внешние сервисы.

Основные цели проекта:

- создание современного AI-чата;
- поддержка различных LLM;
- потоковая генерация ответов;
- расширяемость через MCP;
- дальнейшая интеграция RAG;
- возможность локального хранения истории.

# 7. Работа AI-ассистента

Ассистент получает пользовательский запрос, формирует историю сообщений, добавляет системный промпт и отправляет запрос модели.

Основные этапы:

1. Пользователь вводит сообщение.
2. Формируется ChatRequest.
3. Repository вызывает OpenAI API.
4. Ответ приходит потоково.
5. UI постепенно обновляется.

В будущем между шагами 2 и 3 будет добавлен этап поиска релевантного контекста в локальной базе знаний (RAG).

# 8. AI-компоненты

В проекте используются следующие концепции:

## LLM

Большая языковая модель отвечает за генерацию текста.

## Prompt

Системный промпт определяет поведение модели.

## Context

История сообщений передается вместе с новым запросом.

## Streaming

Ответ отображается пользователю по мере генерации.

## MCP

Позволяет подключать внешние инструменты.

## RAG (планируется)

Позволит искать информацию в локальной базе знаний проекта перед отправкой запроса модели.
