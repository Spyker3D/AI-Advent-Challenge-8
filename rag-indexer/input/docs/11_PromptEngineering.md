## Что такое Prompt Engineering

Prompt Engineering — это процесс проектирования запросов к языковой модели таким образом, чтобы получать наиболее точные, полезные и предсказуемые ответы.

Качество ответа модели во многом зависит от качества сформулированного промпта.

---

## Структура хорошего промпта

Обычно промпт включает:

- роль модели;
- контекст;
- задачу;
- ограничения;
- формат ожидаемого ответа.

Пример:

```text
Ты опытный Android-разработчик.
Используй Kotlin и Jetpack Compose.
Ответ дай пошагово с примерами кода.
```

---

## Основные техники

### Zero-shot Prompting

Модель получает только инструкцию без примеров.

### One-shot Prompting

Перед задачей показывается один пример правильного ответа.

### Few-shot Prompting

Модель получает несколько примеров, после чего решает новую задачу в том же стиле.

---

## System, User и Assistant сообщения

В чат-моделях сообщения имеют разные роли.

**System**

Определяет правила поведения модели.

```text
Ты помощник по Android-разработке.
```

**User**

Содержит запрос пользователя.

```text
Объясни работу Retrofit.
```

**Assistant**

Предыдущие ответы модели, которые сохраняют контекст диалога.

---

## Few-shot Prompting

Пример:

```text
Вопрос: Что такое Activity?
Ответ: Основной компонент Android-приложения.

Вопрос: Что такое Fragment?
Ответ:
```

Модель продолжит в аналогичном стиле.

---

## Chain of Thought

Для сложных задач полезно просить модель рассуждать по шагам.

Пример:

```text
Реши задачу пошагово.
Сначала объясни ход рассуждений,
затем приведи итоговый ответ.
```

Такой подход особенно полезен при анализе данных, программировании и обучении.

---

## Prompt Engineering в Android AI Assistant

В приложении можно использовать:

- системный промпт с ролью AI;
- историю переписки;
- контекст из RAG;
- данные, полученные через MCP;
- текущий запрос пользователя.

Все эти части объединяются в единый запрос к модели.

---

## Best Practices

- Формулируйте задачу максимально конкретно.
- Добавляйте необходимый контекст.
- Ограничивайте формат ответа при необходимости.
- Используйте примеры для повышения качества.
- Не перегружайте промпт лишней информацией.

---

# Итоги

Prompt Engineering позволяет значительно повысить качество ответов языковой модели без изменения самой модели. В Android AI Assistant грамотно составленные промпты вместе с RAG и MCP обеспечивают более точные, последовательные и полезные ответы.
